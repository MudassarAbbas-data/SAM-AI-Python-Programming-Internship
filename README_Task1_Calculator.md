## 📌 Description
A simple command-line calculator that performs basic arithmetic operations. The user inputs two numbers and selects an operation — the program performs the calculation and displays the result.

- Input validation — only accepts numeric values
- Division by zero error handling
- Loop to perform multiple calculations without restarting
- Clean menu-driven interface
